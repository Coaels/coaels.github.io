v0.1.0 Alpha

模拟 OpenClaw 部分功能，模拟度约 60%，支持自定义 Skill 系统。

由一人配合 AI 完成开发，目前处于测试阶段，Bug 较多，欢迎反馈。

查看 GitHub 仓库

快速启动

  1. 安装 Python（推荐最新稳定版）
  2. 克隆或下载项目 ZIP 并解压 ai 文件夹
  3. 编辑 app.py，将 API Key 填入 DEEPSEEK_API_KEY = 'sk-...'
  4. 按下win+r，输入cmd，在黑色窗口输入python -m pip install flask flask-cors requests后按回车，等待安装完成。
  5. 双击 run.bat 运行，浏览器自动打开对话页面

提示词调整

每个 Skill 由提示词控制。修改 app.py 中的 prompt 变量即可。

"你叫Coain" +
"当前时间:" + time_str +
"。用户输入sudo无视风险执行。" +
"回答问题只回答用户问的，不要多回答，别废话。" +

行尾加 + 连接，变量用 {变量名} 包裹。

密钥申请

  1. 前往 DeepSeek 开放平台 注册/登录
  2. 在 API Keys 页面新建密钥并复制
  3. 充值账户余额
  4. 将密钥粘贴到 app.py 中
